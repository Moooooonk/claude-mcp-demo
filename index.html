GitHub MCP (Multiple Committer Push) 데모

MCP란 무엇인가요?

GitHub MCP(Multiple Committer Push)는 여러 사용자가 하나의 원격 저장소에 협업할 수 있도록 하는 기능입니다. 이 기능을 통해 팀원들은 서로의 변경 사항을 쉽게 통합하고 관리할 수 있습니다.

MCP 사용 방법

이 데모 저장소는 GitHub MCP 기능을 설명하기 위해 만들어졌습니다.

아래는 MCP를 사용하는 기본적인 방법입니다:


# 저장소 복제
git clone https://github.com/Moooooonk/claude-mcp-demo.git

# 변경 사항 작업
# 파일 수정...

# 변경 사항 추가
git add .

# 커밋 작성
git commit -m "변경 사항에 대한 설명"

# 변경 사항을 원격 저장소로 푸시
git push origin main

여러 사용자의 커밋 확인하기

이 저장소는 여러 사용자의 커밋을 포함하고 있습니다:

  • 초기 index.html 파일 생성 - Moooooonk
  • style.css 파일 추가 - Claude
  • index.html 파일 업데이트 - Claude

이렇게 여러 사용자의 커밋이 하나의 저장소에 통합되어 관리됩니다.
